How to use the template system

In order to allow you to add only the sides of the showcase that you want, this tool uses a system which requires you to do some small modifications to your showcase:

Color Code

This system involves a color code in which pixels of certain colors determine where you want a certain side of your showcase to appear (like the side, front or back). You will have to modify your template to include pixels with these colors that are located wherever you want that side to apppear. Keep in mind that no pixel can have these specific RGB values other than these ones.

  1. rgb(255, 0, 0) - Right side
  2. rgb(0, 255, 0) - Front
  3. rgb(0, 0, 255) - Back
  4. rgb(255, 255, 0) - Left side

Note

Since this is hosted on Github I can't have the showcase download automatically or add a button that starts the download. You will have to manually right click and "save as" in order to download your showcase.
